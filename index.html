Privacy Policy for Neo Connect

Last updated: 2026-03-02

Neo Connect is a client application designed to establish secure connections to private servers using user-provided access credentials.

Data Collection

Neo Connect does not collect personal data.

The application does not track users and does not sell or share user information.

Network Processing

All network processing occurs locally on the user's device to establish encrypted connections to user-provided servers.

The developer does not receive or store connection logs on external servers.

Third Parties

Neo Connect does not transmit personal data to third parties.

Contact

If you have questions about this Privacy Policy, contact: support@neoconnect.app
